# Meand You

Welcome to the Meand You project repository. This is a landing page application for the Meand You brand.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to work locally:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Making Changes

- You can edit the code in your preferred IDE and view the changes on your local development server.
- The entry point for the application is `src/main.tsx` and the main component is `src/App.tsx`.
